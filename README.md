
# API PARKING SPOT

API Voltada para cadastramento de veículos em vagas de estacionamento de hotel.


## Documentação da API

#### Retorna todos os itens

```http
  GET /parking-spot
```

#### Retorna um item

```http
  GET /parking-spot/{id}
```

#### Adiciona um item

```http
  POST /parking-spot/{id}
```

#### Deleta um item

```http
  DELETE /parking-spot/{id}
```

#### Modificar um item

```http
  PUT /parking-spot/{id}
```
